## E-Commerce Sales Analysis Dashboard

## Project Overview

In this project, I analyzed an e-commerce dataset to understand how sales and profit are performing across different regions and product categories. The goal was to take raw data and turn it into useful insights that can help businesses make better decisions.

---

## What I Did

* Explored the dataset to understand the data
* Cleaned the data by removing duplicates and handling missing values
* Used SQL to analyze sales and profit
* Created visualizations to understand trends
* Built an interactive dashboard using Power BI

---

## Tools Used

* Python (Pandas, Matplotlib, Seaborn)
* SQL (PandasQL)
* Power BI

---

## How I Worked on the Project

### 1. Data Collection

I used a dataset from Kaggle for this project.

### 2. Data Cleaning

I made sure the data was clean by removing duplicate rows and fixing any missing values so that the analysis would be accurate.

### 3. Data Analysis

I used SQL queries to find:

* Total sales
* Sales by region
* Profit by category
* Sub-categories that are making losses

### 4. Dashboard Creation

Finally, I created a Power BI dashboard to present all the insights in a clear and interactive way.

---

## Dashboard Features

* KPI cards showing Total Sales, Total Profit, and Total Quantity
* Sales by Region
* Profit by Category
* Sales vs Profit comparison
* Sub-category analysis
* Filters to explore data by Region and Category

---

## Key Insights

* The West region has the highest sales
* The Technology category gives the most profit
* Some sub-categories are causing losses
* High sales do not always mean high profit
